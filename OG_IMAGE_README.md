# OG Image Placeholder

This directory needs an `og.png` file (1200x630px recommended).

The layout.tsx metadata references `/chargeback-brief/og.png` for OpenGraph
and Twitter card previews.

## Quick creation options:
- Use Figma / Canva to make a 1200x630 branded card
- Use https://og-image.vercel.app/ as a starting point
- Keep it simple: dark background, "Chargeback Brief" title, tagline

Drop the final file here as `og.png` and it will be served automatically.
